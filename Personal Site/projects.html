ACADEMIC PROJECTS

  • Common Disease Predictor

    • SSN College of Engineering | January, 2019

    Engineered and built a deep learning based detector using TF, Keras & Flask which predicts common diseases that display their symptoms externally like eye, skin, mouth & nail diseases by sending pictures from a mobile App to a Flask Web Server.

  • Accident Detection and Advanced Traffic Clearance

    • YAH Hackathon, ACM | September, 2019

    • Built a vehicular crash detection module which uses impact sensors & GPS module (Neo-6m) to detect an accident and report GPS co- ordinates to the nearest ambulance driver through a flutter based mobile app.

    • Built a smart traffic control system which detects ambulances in traffic signal junctions using YOLOv3 object detection algorithm and makes way for the ambulance to move further without any hindrance. In addition to this med-cloud, a web app which uses fingerprint scanning was developed to retrieve patient medical history.
